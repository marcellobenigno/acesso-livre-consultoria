empresa

A Acesso Livre Consultoria é uma empresa do ramo de Geotecnologias que utiliza no desenvolvimento de suas soluções Softwares Livres e de Código-Aberto. A nossa área de atuação concentra-se em três principais áreas: desenvolvimento de soluções WebGIS, implantação e suporte de Sistemas de Informações Geográficas (SIG) e capacitação profissional.

Conheça Nossos Cursos


  • QGIS
    O QGIS é um dos mais completos softwares de SIG, possui diversos plugins que aumentam deixam ainda mais os seus recursos.
  • GRASS
    O GRASS é um dos mais antigos e poderosos softwares de SIG, oferece uma série de ferramentas para análises de dados raster e vetorial.
  • PostgreSQL e PostGIS
    Quando se trata de armazenamento de dados espaciais, o SGBD PostgreSQL e a extensão espacial PostGIS oferecem uma das melhores soluções.
  • Desenvolvimento de aplicações WebGIS com Python e GeoDjango
    Python e Django proporcionam o desenvolvimento rápido de aplicações Web. Somado este poder ao GeoDjango o desenvolvimento de aplicações WebGIS ganha ainda mais qualidade.
  • GeoServer
    O GeoServer é um servidor de dados espaciais que possibilita a geração de Web Services OGC.
  • MapServer
    O MapServer é um software livre que serve como ambiente de desenvolvimento para construção de aplicativos espaciais na internet.
  • p.mapper framework
    Baseado no MapServer, o p.mapper é um poderoso framework WebGIS.
  • Leaflet.js
    O Leaflet.js é uma biblioteca javascript ideal para se trabalhar com o GeoServer ou o GeoDjango, bastante flexível e cheia de plugins.
